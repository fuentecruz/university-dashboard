# University Student Dashboard

## Descripción
Este proyecto consiste en el análisis de datos universitarios y la creación de un dashboard interactivo utilizando Streamlit.

## Objetivo
Analizar la retención y satisfacción estudiantil para identificar patrones y apoyar la toma de decisiones.

## Dataset
El dataset incluye información sobre:
- Año académico
- Periodo (Spring/Fall)
- Aplicaciones, admitidos y matriculados
- Tasa de retención
- Nivel de satisfacción
- Distribución por áreas académicas

## Tecnologías
- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit

## Dashboard
Permite:
- Filtrar por año y periodo
- Visualizar métricas clave (KPIs)
- Analizar tendencias de retención y satisfacción
- Ver distribución de estudiantes por área
